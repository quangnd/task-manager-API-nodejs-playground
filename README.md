# Task Manager API 🗒

## Features

- CRUD operations. In associate with MongoDB.
- Authentication and protect API with middleware.
- File upload.
- Simple sort, filter and paging.
- Test with Jest .

## Up & Running 🏃‍♂️

1. Run `npm install`
2. Rename `.env_example` to `dev.env`
3. Run in development `npm run dev`
4. Test `npm test`
